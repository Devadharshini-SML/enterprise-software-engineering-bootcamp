# Day 1 Reflection

## What responsibilities do enterprise engineers have?
Enterprise engineers design, develop, test, and maintain large-scale applications. They ensure the system is secure, scalable, and efficient. They also work in teams and follow structured development processes.

## What types of systems do they build?
They build systems like banking applications, e-commerce platforms, healthcare systems, cloud-based services, and enterprise automation tools.

## Role of software engineers in enterprise systems

Software engineers play a key role in building and managing enterprise systems. They analyze requirements, design system architecture, write clean and efficient code, and ensure system performance. They work closely with other teams like testers, designers, and managers to deliver high-quality software. They also ensure security, scalability, and smooth functioning of applications.

## Developer tooling ecosystem

The developer tooling ecosystem includes all the tools used by developers to build, test, and manage software. This includes code editors like VS Code, version control systems like Git, collaboration platforms like GitHub, testing tools, containerization tools like Docker, and API tools like Postman. These tools help developers work efficiently, maintain code quality, and collaborate in teams.

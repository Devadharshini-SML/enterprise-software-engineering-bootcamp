# Day 2 - SOLID Principles

## Why SOLID improves maintainability
SOLID principles help developers write clean and modular code. Each class has a single responsibility, which makes the code easier to understand, test, and modify. It reduces complexity and improves scalability.

## Examples in large systems
In large systems like e-commerce or banking applications, different services handle specific tasks such as payment, user management, and notifications. SOLID principles help teams work independently and update systems without affecting other parts.
